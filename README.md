# How to start:
1. Download repository
2. Create an env
    ```sh
    python -m venv env
    ```
3. Activate env
    ```sh
    source env/bin/activate
    ```
2. install requirements
    ```sh
    pip install -r requirements.txt
    ```
3. Run scripts 
    ```sh
    python [file-name].py
